# Kendo Video Analyzer
剣道歴16年の知見を活かした、AIによる打突シーン自動検知アプリ

##  概要
本プロジェクトは、剣道の試合や稽古の映像から「面・小手・胴」の打突シーンをAIで自動抽出し、客観的な分析を可能にするWebアプリケーションです。
指導者の主観に頼りがちな評価をデータ化し、自身のパフォーマンス向上を支援します。

## 主な機能
- **動画解析**: Amazon Rekognitionを用いた高度な動体検知
- **打突検知**: 高速な剣道の動きから有効打突シーンを特定
- **可視化**: 分析結果をグラフや統計データとして表示

## 技術スタック
- **Language**: Python
- **Framework**: Streamlit
- **Cloud**: AWS (S3, Amazon Rekognition)
- **Library**: Pandas, Boto3
