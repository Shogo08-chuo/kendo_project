# Kendo Video Analyzer

剣道の試合動画から「面・小手・胴」の打突を自動検出し、分析を支援するAIアプリケーション

## Overview
本プロジェクトは、剣道の試合・稽古動画を解析し、打突（面・小手・胴）を自動検出することで、パフォーマンス分析を支援するアプリケーションである。  
従来、剣道の試合分析は指導者の経験や主観に依存していたが、本システムにより打突の発生回数や傾向を定量的に把握することが可能となる。

## Problem
剣道の試合分析には以下の課題が存在する：

- 打突の判定や評価が指導者の主観に依存している  
- 試合動画の分析に時間がかかる（手動での確認）  
- データとして蓄積されず、戦術改善に活かしづらい  

## Solution
本システムは以下の流れで課題を解決する：

1. 試合動画をアップロード（S3）  
2. Amazon Rekognitionにより動画内の動体・人物を検出  
3. 打突動作の特徴から「面・小手・胴」を分類  
4. 打突回数や発生タイミングを可視化  

## Key Features

### Video Processing
AWS S3にアップロードされた動画を対象に解析を実行  

### Strike Detection
Amazon Rekognitionを活用し、剣道特有の高速動作から打突シーンを抽出  

### Visualization
打突回数をグラフとして可視化し、分析を容易にする  

## Architecture
本システムは以下の構成で動作する：

- フロントエンド: Streamlit  
- ストレージ: AWS S3  
- 解析: Amazon Rekognition  
- 処理: Python（Boto3経由でAWS連携）

## Result
- F1 Score: 約0.62（打突検知）  
- データ数: 約50動画（現時点）  

※データ数の制約により精度は改善途中

## Challenges
本プロジェクトでは以下の課題に直面した：

- 剣道特有の高速動作により検出が不安定  
- 学習データの不足（50件程度）  
- 打突の定義（有効打突か否か）の曖昧さ  

## Future Work
- データ数の拡張による精度向上  
- 独自モデル（YOLO等）への移行  
- UI/UX改善による実用化  
- リアルタイム解析への対応
